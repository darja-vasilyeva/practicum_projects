## Описание проектов

Эти проекты были выполнены в ходе обучения в Яндекс.Практикуме (2024-2025).

| №  | Название                      | Описание                                                                                                                                                                                                                  | Технологии                                                     |
|----|-------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------|
| 01 | SWAPI                         | С помощью клиента для публичного API получить информацию о персонажах, планетах, звездолётах и других объектах вселенной «Звёздных войн» в формате JSON.                                                                    | ООП, HTTP API, обработка исключений, сохранение данных на диск |
| 02 | GastroHub                     | Создать базу данных для управления сетью ресторанов, включая информацию о меню, заказах и клиентах; выполнить аналитические запросы.                                                                                        | SQL: оконные функции, вложенные подзапросы, фильтрация, агрегации |
| 03 | DWH для нескольких источников | Создать хранилище данных, способное интегрировать и обрабатывать информацию из различных источников. <br/> Задачи: <br/> 1) интеграция данных (PostgreSQL, MongoDB); <br/> 2) очистка и трансформация; <br/> 3) загрузка в DWH; <br/> 4) анализ и отчётность. | PostgreSQL, MongoDB, API, Python, SQL, Airflow, DAGs           |
